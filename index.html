Firefox Dev

Firefox Developer Edition é um navegador livre e multiplataforma desenvolvido pela Mozilla Foundation. A intenção é um navegador para feito desenvolvimento web e altamente extensível. Possui ótimas features para visualização e depuração de documentos web.

OpenDesktop

Uma plataforma livre com multiplas funcionalidades e repleta de extensões. Também hospeda o OpenCode.net que é uma plataforma web de desenvolvimento e repositórios. Além de ser um provedor Nextcloud fazendo uso de todo seu ambiente desde notas a gerenciamento de projetos com o Deck.

GitLab

GitLab é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, cada diretório não dependente de acesso a uma rede ou a um servidor central. Uma alternativa ao GitHub que possui integração direta com o OpenCode e o OpenDesktop muito propício para o nosso caso.

O que é "Software Livre"?

O software livre é um tipo de software desenvolvido e mantido de maneira colaborativa, onde o código-fonte está disponível para todos que quiserem usar, investigar, alterar e redistribuir. O open source beneficia diferentes iniciativas, pois não está focado em obter lucros relacionados à propriedade intelectual.Isso contrasta com os aplicativos proprietários ou softwares proprietários, por exemplo, o Microsoft Word e o Adobe Illustrator, que são vendidos aos usuários finais pelo criador ou portador do copyright e não podem ser editados, aprimorados ou redistribuídos exceto quando especificado pelo portador do copyright.
